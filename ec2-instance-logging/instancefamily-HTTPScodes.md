#Series	
C – Compute optimized
D – Dense storage
F – FPGA
G – Graphics intensive
Hpc – High performance computing
I – Storage optimized
Im – Storage optimized (1 to 4 ratio of vCPU to memory)
Is – Storage optimized (1 to 6 ratio of vCPU to memory)
Inf – AWS Inferentia
M – General purpose
Mac – macOS
P – GPU accelerated
R – Memory optimized
T – Burstable performance
Trn – AWS Trainium
U – High memory
VT – Video transcoding
X – Memory intensive
Z – High memory

# Options
a – AMD processors
g – AWS Graviton processors
i – Intel processors
m1ultra – Apple M1 Ultra chip
m2 – Apple M2 chip
m2pro – Apple M2 Pro chip
b – Block storage optimization
d – Instance store volumes
e – Extra storage or memory
flex – Flex instance
n – Network and EBS optimized
q – Qualcomm inference accelerators
*tb – Amount of memory for high-memory instances (3 TiB to 32 TiB)
z – High CPU frequency
